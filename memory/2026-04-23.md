## 15:35 | Distinguished fixed-subset significance from subset-selection stability

- kind: decision
- tags: [rsa, permutation, subspace-search, interpretation, validation]
- topic: rsa-workflow
- source: user question + assistant clarification
- promotion_hint: promoted
- related_state: [state/heartbeat-state.json]
- related_memory: [MEM-20260423-001]

Clarified that a chemical-subset sampled robustness check and permutation test
can support reliability, but only for the claim actually tested. If the chosen
subset is frozen and then re-evaluated across resamples or label shuffles, that
supports conditional robustness of the fixed subset. If subset selection is
rerun inside each resample or permutation, that instead supports selection
stability and search-corrected significance. Out-of-sample date or stimulus
validation is still a separate generalization claim.
