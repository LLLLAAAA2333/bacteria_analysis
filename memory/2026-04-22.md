Changed stimulus reliability tiers from view-wise quantile labels to absolute thresholds. A stable incident pair is now `rank_stability >= 0.80`; high stimulus reliability requires `mean_rank_stability >= 0.85` and `stable_pair_fraction >= 0.50`; medium requires `mean_rank_stability >= 0.80` and `stable_pair_fraction >= 0.35`; supported stimuli below those thresholds are low.
